---

## 📌 Project Overview

Developed an AI-powered portfolio website featuring a conversational virtual assistant that enables visitors to learn about my professional experience, technical skills, and projects through real-time interactions. The application provides a responsive and intuitive user interface with support for both English and Hebrew conversations.
